Blog yang dikelola oleh [adeka](adeka.id/) yang akan digunakan sebagai medium untuk berbagi tentang tips teknologi forntend dan desain. Dan juga digunakan menaruh portofolio dari beberapa projek yang kami kerjakan.

## Stack
Situs ini dibangun menggunakan beberapa stack :

**Nextjs** sebagai core untuk blog ini

**MDX** untuk custom markdown pada tulisan terkait

**Vercel** tempat deploy blog ini

**Github** menaruh source
